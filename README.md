# 🚀 Industrial IoT & Full Stack Portfolio | Roberto Drazewski

¡Bienvenido a mi portfolio profesional! Soy **Técnico Electrónico** especializado en el desarrollo de soluciones que conectan el hardware con el software. Este proyecto no solo muestra mi trabajo, sino que refleja mi enfoque en la ingeniería de sistemas escalables y robustos para el sector industrial.

## 🛠️ Tecnologías Utilizadas

### **Hardware & Firmware**
* **Languages:** C++, Python, MicroPython.
* **Core:** Embedded Systems, IoT Industrial, Circuit Design.

### **Full Stack Development**
* **Frontend:** React, Vite, Tailwind CSS (v4), TypeScript.
* **Backend:** Node.js, Express.
* **Databases:** MySQL.
* **Testing:** Selenium, Postman.

### **Infrastructure & Tools**
* **Localization:** i18next (Soporte Multi-idioma ES/EN).
* **Automation:** GitHub Actions.
* **Design:** Industrial Glassmorphism UI.

---

## 🌟 Características Destacadas

- **Sincronización Dinámica con GitHub:** El portfolio consume automáticamente mis repositorios públicos mediante la API de GitHub.
- **Arquitectura Multi-idioma:** Sistema de internacionalización completo para alcance global.
- **Diseño Responsive & Industrial:** UI optimizada para máxima claridad en sistemas críticos y dispositivos móviles.
- **Hero Interactivo:** Incluye una demostración en video de mis capacidades técnicas integradas.

```text

🚀 Cómo ejecutarlo localmente
Clona el repositorio:

Bash
git clone https://github.com/RobertoDrazewski/portfolio.git

Instala las dependencias:

Bash
npm install
Inicia el servidor de desarrollo:

Bash
npm run dev
